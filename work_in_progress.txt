# Guitar Rhythm Trainer - Work in Progress

## Текущее состояние проекта

Мы разрабатываем приложение Guitar Rhythm Trainer, которое позволяет музыкантам тренировать чувство ритма при игре на гитаре. Приложение использует аудиоинтерфейс для захвата звука гитары, анализирует его и предоставляет обратную связь о точности попадания в ритм.

### Основные файлы проекта:

1. **main.py** - главный файл приложения с интерфейсом на Kivy
2. **audio_processor.py** - модуль для обработки аудио с использованием PyAudio
3. **rhythm_analyzer.py** - модуль для анализа ритма
4. **requirements.txt** - зависимости проекта
5. **INSTALL.md** - инструкция по установке и использованию
6. **click.wav** - звук метронома

### Последние изменения:

1. Обновили audio_processor.py для использования PyAudio вместо sounddevice
2. Создали rhythm_analyzer.py для анализа ритма
3. Обновили main.py для интеграции с новыми модулями
4. Добавили поддержку выбора входного канала аудиоинтерфейса
5. Добавили функцию мониторинга звука (воспроизведение звука гитары через выбранное устройство вывода)
6. Обновили INSTALL.md с информацией о новых функциях

### Текущие проблемы:

1. Необходимо установить PortAudio для работы PyAudio (выполнено через brew install portaudio)
2. Установлены зависимости из requirements.txt
3. Скачан файл click.wav для звука метронома
